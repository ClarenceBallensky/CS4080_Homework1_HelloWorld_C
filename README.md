# My first C program
Defines a doubly linked list of heap-allocated strings. Includes functions to insert, find, and delete items from the list.

## How to run
Prerequisites: install a GCC on your machine to compile C code, install an IDE on your machine that supports C
1. Clone this GitHub repository on your machine
2. Open up the terminal
3. Type the command "gcc HelloWorld.c -o Hello"
4. Type the command "./Hello"
